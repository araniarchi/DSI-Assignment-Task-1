# DSI-Assignment-Task-1
For the purpose of the task, 4 classes were created which are as following,

OOP is the main class which starts the program. 
Control class takes natural number inputs for specific methods such as add, buy, sell or delete products which are implemented in ProductOption class. 
Product class has the necessary attributes and functions of a product.
ProductOption class contains the implementation details of add, buy, sell, delete and show products and of the available balance.
